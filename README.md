# SOF 2026

Projeto desenvolvido com **Python e Django**.

## 📌 Descrição
SOF é um projeto web desenvolvido para estudo e prática utilizando o framework Django.

## 🚀 Tecnologias utilizadas
- Python
- Django
- HTML / CSS
- SQLite (padrão do Django)

## 📂 Estrutura do projeto
sof/
- venv/ (ambiente virtual)
- manage.py
- app/
- templates/
- static/

## ⚙️ Como instalar

Clone o repositório:

git clone https://github.com/clovis2025/sof.2026.git

Entre na pasta:

cd sof.2026

Crie ou ative o ambiente virtual:

python -m venv venv

Ativar no Windows:

venv\Scripts\activate

Instale as dependências:

pip install -r requirements.txt

## ▶️ Executar o projeto

python manage.py runserver

Depois abra no navegador:

http://127.0.0.1:8000/
